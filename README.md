# Heart Disease Prediction using Machine Learning

## Project Overview

This project builds a machine learning model to predict whether a person is at risk of heart disease based on medical attributes.

The model is trained using the **Heart Disease UCI Dataset** and uses classification algorithms to identify patterns that indicate heart disease risk.

## Dataset

Dataset: Heart Disease UCI Dataset
The dataset contains medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Cholesterol Level
- Resting Blood Pressure
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression

Target variable:
1) `0` → No heart disease
2) `1` → Heart disease risk


## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Feature Importance Analysis

## Machine Learning Model

The model used in this project:

1) Logistic Regression

The model predicts whether a patient has heart disease risk based on their health data.

## Model Evaluation

The model is evaluated using:

1) Accuracy
2) Confusion Matrix
3) ROC Curve

These metrics help measure the performance of the classification model.

## Key Insights

Feature importance analysis highlights the most influential medical attributes affecting heart disease prediction.

Important factors include:

1) Chest pain type
2) Maximum heart rate
3) ST depression
4) Exercise induced angina
